Web development ნიშნავს ვებსაიტების შექმნას. ეს პროცესი ორ ნაწილად იყოფა: პირველი არის frontend, რაც იმას ნიშნავს, რომ ეს არის ის, რაც ჩვენ ვხედავთ ვებსაიტზე. ამისთვის გვჭირდება HTML, CSS და JavaScript. მეორე არის backend, რაც ნიშნავს, რომ ეს არის ის ნაწილი, რაც არ ჩანს. აქ პროგრამები მუშაობენ, როგორიცაა Python და PHP. ვებსაიტის შექმნის პროცესი იწყება იმით, რომ ვგეგმავთ, რა უნდა იყოს საიტზე, შემდეგ ვამუშავებთ დიზაინს, ვწერთ კოდს, ვამოწმებთ, შემდეგ ვაგრძელებთ ინტერნეტში და ბოლოს მუდმივად ვახლებთ. ეს ყველაფერი ერთად ქმნის ვებსაიტებს, რომლებსაც ყოველდღიურად ვიყენებთ.
